A high-performance, visually stunning portfolio website showcasing software engineering, security, and hardware hacking expertise. This project focuses on modern aesthetics, fluid motion, and robust full-stack integration.

---

## ✨ Features

- 📱 **Fully Responsive** - Optimized for every screen size, from mobile to ultra-wide displays.
- 🎯 **Scroll Snapping** - Intuitive vertical navigation providing a focused, slide-like experience.
- 🎭 **Framer Motion Animations** - Sophisticated UI transitions and interactive elements for a premium feel.
- ⌨️ **Dynamic Typewriter** - Engaging hero section highlighting multiple technical roles.
- 🎨 **Modern Dark UI** - Glassmorphism effects, custom gradients, and a sleek dark-mode aesthetic.
- 🔍 **Advanced SEO** - Comprehensive metadata, JSON-LD schema, OpenGraph, and Twitter tags for maximum visibility.
- 🚀 **Performance Optimized** - Leverages Next.js 16 App Router and Tailwind CSS 4 for industry-leading speed.

## 🛠️ Tech Stack

### Frontend
- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) (Vanilla CSS preference)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)

### Tools & Deployment
- **Platform:** [Vercel](https://vercel.com/)
- **Linting:** [ESLint](https://eslint.org/)
- **Version Control:** Git & GitHub

## 🚀 Installation & Setup

### Prerequisites

- **Node.js:** 20.x or higher
- **npm** (or pnpm/yarn)

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/qwzynx/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
